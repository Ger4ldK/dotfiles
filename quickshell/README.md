# Quickshell widgets
This will be tedious and probably not as fun but hopefully the end results are nice.
I thought I left frontend work behind ;-;

## Checklist
- System diagnostics (RAM usage, CPU usage, temps, network speed, etc.)
- I/O management (USB, Display, BT, Wi-Fi, Audio)
- Info (Date/ Calendar)
- Styling (cwal setup is done, but colors need to be picked up from the `Colors.qml` file)
- Dynamic behaviors (pop up on hover, hide after x seconds, etc.)
- Wallpaper picker with cwal call after
- App launcher
- Power controls (shutdown, restart)

## Mindset for widgets
- If I frequently need to open my terminal to check on it/adjust it, I'm making it a widget
